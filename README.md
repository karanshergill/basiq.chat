# Hardware
- Minimum 12GB RAM (16GB+ preferred)
- Quad-Core CPU
- 80GB+ SSD

# OS
- Ubuntu 24.04.2 LTS (Noble Numbat)

# Core Tools
- git, curl, unzip, make, build-essential, python3, g++

# Installation

```shell
sudo apt update
```
```shell
sudo apt install git curl unzip make build-essential python3 g++
```


```shell
usermod -aG sudo karan
```
```shell
su - karan
```
